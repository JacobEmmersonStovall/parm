Hello there!

This site is powered by the same source code that powers parm.app. Everything in this box is inside a "node". You can enter whatever you want in this node, including markdown, React components, and more.

The website is even partially built via these nodes. For instance, the navigation menu in the sidebar is handled via a special named node called navigation. Here is the source text for that node:

<ListItem
  button
  component="a"
  aria-label="Create"
  href="/?focus=create"
>
  Create
</ListItem>

There's lots of different custom React components that are allowlisted for use in a node. For example, <DateSubtracter/> is one that is easy to use because it does not require any props. However, some custom React components do require props. To learn what components are available (and how to use them) I recommend checking out the source code of the Markdown component here.

By the way, you can even see the DateSubstracter in action right here!
